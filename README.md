# Fret Department

**Fret Department** is little vibe coded web app for viewing
guitar chords and scales in terms of fretboard positions.
Implemented in pure HTML/JavaScript/CSS with no external
dependencies, it is available as a single self-contained HTML
file that can be run directly from disk with no HTTP server
required.

I'm sure you're familiar with at least some of the issues
around vibe coding. I'm ambivalent about it, but need to
understand it for work reasons. And it's a lot of fun, up
to a point. I have little patience for the breathless boosters
and I wouldn't want my life to depend on anything written this
way, but it's great for carefully circumscribed low stakes
recreational nonsense like this.
